== Description ==
[Demo](http://quadmenu.com/total/) | [Premium](http://quadmenu.com) | [Documentation](http://quadmenu.com/documentation/) | [QuadMenu](https://wordpress.org/plugins/quadmenu/)

Integrates QuadMenu with the [Total Theme](https://themeforest.net/item/total-responsive-multipurpose-wordpress-theme/6339019?ref=quadlayers). Requires QuadMenu and Total Theme.

This plugin will add all the settings to match the style and the layout of Total theme.

== Screenshots ==

Demo Site: http://quadmenu.com/total/

1. QuadMenu Mega Menu working with Total
2. This plugin adds a preconfigured "Total" theme to fit QuadMenu Mega Menu settings.

== Installation ==

1. Go to the Plugins page in WordPress
1. Search for "Total Mega Menu"
1. Click "Install"

After installation go to Appearance > Menus and create your mega menu.
